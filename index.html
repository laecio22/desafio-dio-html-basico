Utilizando tags de mídias em Html

O que são mídias no Html?

Mídias São os vídeos , imagens e aúdios adicionados na estrutura Html, eles dão ao site uma cara mais tecnológica.

Principais exemplos de tags de mídias

  1. img responsável por mostrar uma imagem na tela, possui os atributos src que indica o endereço da imagem e o alt que fornece uma descrição da imagem para os leitores de tela.
  2. audio responsável por mostrar aúdios na tela, possui atributo controls que mostra os controles padrões e uma tag filha source que com o src neles indica o caminho do arquivo de música.
  3. video responsável por mostrar vídeos na tela, possui atributo controls que mostra os controles padrões e uma tag filha source que com o src neles indica o caminho do arquivo de vídeo.
  4. iframe Faz com que seja integrado no seu site o contéudo de um site externo, por exemplo um vídeo do youtube.

Criando tabelas em Html

O que são tabelas no Html?

tabelas são uma forma de estruturar informações no html, através de linhas e colunas, apesar ser utilizada anteriormente como forma de estruturar o layout do site, isso não é uma boa prática.

Principais exemplos de tags de tabelas

  1. table cria a tabela em si, possui os atributos border para criar a borda da tabela, width para definir a largura da tabela, align para alinhar a tabela na tela.
  2. tr responsável por criar linhas na tabela
  3. td responsável por criar colunas na tabela
  4. th dá uma formatação especial na linha da tabela

Trabalhando com formulários em Html

O que são formulários no Html ?

formulários é uma estrutura que o html fornece que permite que o usuário envie dados do cliente para o servidor, utiliza os inputs para que o usuário forneça dados.

Principais exemplos de tags do formulário

  • form cria o formulário, possui os atributos action cujo valor fornecido, indica para onde os dados do formulário estão sendo enviados . method que indica a forma que estão sendo enviados os dados, por exemplo: GET informa que está enviando pela url e o POST que envia no body de uma requisição.
Principais inputs do formulário
  • input type='text' Permite que o usuário digite valores de string no campo.
  • input type="number" Permite somente valores number podendo ter um valor minimo com o atributo min e um valor máximo com max
  • input type="date" Permite escolher um valor de data
  • input type="file" Permite escolher um arquivo
  • input type="email"O valor digitado tem que obedecer o padrão de email

HTML Semântico e acessibilidade

O que é o Html semântico?

semântica no html São as melhores práticas em Html, em relação a estrutura e nomenclatura de forma que tenha significado bem definido, facilitando com seu site tenha boa classificação de SEO.

Principais exemplos de tags semânticas

  • main indica que é a tag principal de uma página e que só pode ter uma dela por página e que não pode está sendo envolta por outras tags como section e article.
  • section essa tag cria sessões na página , permitindo uma divisão mais estruturada.
  • nav cria uma área de navegação.
  • header indica que é o cabeçalho .
  • footer indica que é o rodapé
